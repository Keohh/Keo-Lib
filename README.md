# Keo-Lib
- Collection of work based on previous assignments and passion projects.

## Contents
- K-Assort
    - Assortment of Dev Tools and Server Functionality Options like vehicle rewards and weapons. tldr; a group of client based scripts complied into a libary resource
- K-Crime
  - Tire pop script
- K-Cuffing
   - Econ styled cuff script using PS-UI to allow cuff breaking on standalone styled servers. Used interact sound and callback features to give a more rich setting
- K-Jailing
   - Simple jailing script using time(months(seconds)) and teleports the player into prison. Ties in with cuffing to add a failsafe if the player forgets to uncuff their arrestee
- K-JStatus
  - Drawtext 2D text at the top right portion of the screen to display if Emergency Services and or Tow is on duty. Using a json file for status saving across server restarts
- K-Prio
  - Prio request system using json for storage saving to allow players to grab status while joining for no confusion. Uses commands and staff approval for status changing
- K-Radioanim
  - Radio Animation script using [Big-Daddy's Radio Animations](https://bigdaddyscripts.com/Products/View/2196/Radio-Animation). Using commands the player can change their radio animation on the fly and saves to their cache for their next playthrough.
